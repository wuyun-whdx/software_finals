性格雷达·生活指南

Vue 3 Spring Boot MySQL Cloudflare Tunnel

面向个性化生活建议的完整前后端系统，通过性格测试、偏好答题和规则推荐，把用户画像转化为餐饮、旅行、穿搭与生涯规划建议。

  • 注册登录与 JWT 权限控制：区分普通用户与管理员，保护后台管理能力。
  • 性格与偏好测试：内置基础性格、饮食偏好、旅游偏好题库，自动计算五维画像分数。
  • 雷达图画像报告：展示开放性、尽责性、外向性、宜人性、神经质等维度，并生成文字解析。
  • 场景化推荐与反馈优化：根据用户画像和反馈权重，动态调整餐饮、旅行、穿搭、生涯推荐排序。
  • 双人适配与分享：支持通过好友手机号生成适配报告，并创建只读分享链接。
